  • What Happens When You Enter a URL into a Browser?
    • Describe the steps involved, from entering a URL to seeing the web page
    1. Open your web browser
    2. Hit ctrl+l on your keyboard or click the long box on the upper portion of the browser
    3. Type www.google.com followed by Enter
    4. The url makes a request to the google.com server by filling out the header with named pairs such as Authorization: Bearer Token
    5. The server sends a response to the ip's browser
    6. The browser now renders a response to the webpage at google.com
